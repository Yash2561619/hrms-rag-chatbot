OPTION A: Test locally (5 minutes, no WhatsApp needed)
────────────────────────────────────────────────────

  Step 1: Install
    pip install -r requirements.txt

  Step 2: Set API key
    export ANTHROPIC_API_KEY="sk-ant-xxxxx"

  Step 3: Run demos
    python step1_extract.py      # PDF extraction
    python step2_chunk.py        # Semantic chunking
    python step3_embed.py        # Embeddings
    python step4_store.py        # Vector database
    python step5_query.py        # Full RAG pipeline ← Best one!
    python step6_wrong_question.py  # Hallucination prevention

  Each step shows ✓ success messages.
  This teaches you how RAG works!

OPTION B: Deploy with WhatsApp (5 minutes)
────────────────────────────────────────────────────

  Step 1: Install & configure
    pip install -r requirements.txt
    cp .env.example .env
    # Edit .env with Anthropic API key

  Step 2: Start Flask server (Terminal 1)
    python app.py

  Step 3: Start Cloudflare tunnel (Terminal 2)
    cloudflared tunnel --url http://localhost:5000

  Step 4: Configure webhook
    • Copy Cloudflare URL from Terminal 2
    • Go to Meta Developer Dashboard → WhatsApp
    • Set Webhook URL + Verify Token
    • Test by sending message to bot

  See QUICKSTART.md for detailed steps.

❓ FAQ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: Do I need a WhatsApp business account?
A: Only if you want WhatsApp integration. You can test everything locally!

Q: What if I don't have a PDF yet?
A: The system uses demo chunks automatically. Works perfectly for testing.

Q: How much does this cost to run?
A: ~$0.01 per query (Claude Opus), <$10/month infrastructure

Q: Can I use a different LLM?
A: Yes! Replace claude.messages.create() with OpenAI, Cohere, or others

Q: How do I add more HR policies?
A: Replace SAMPLE_HR_POLICY.txt and restart Flask (auto-rebuilds index)

Q: What if the AI hallucinates?
A: It won't! System prompt forces grounding in policy text only

Q: Can I run this on AWS/GCP/Azure?
A: Yes! See SETUP_GUIDE.md section 7 for AWS EC2 deployment

"ANTHROPIC_API_KEY not found"
  → cp .env.example .env

"No messages received from WhatsApp"
  → Check: Flask running? Tunnel running? Webhook URL correct?
  → See SETUP_GUIDE.md section 8 for detailed troubleshooting

"Token expired"
  → You're using a 24-hour token. Get permanent System User token instead.
